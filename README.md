# Connor Colyer Site v2 (Vite + React)

## Dev
```bash
npm i
npm run dev
```

## Deploy (GitHub Pages)
If this repo is **connorcolyer.github.io** (user site): keep `base: "/"` in `vite.config.js`.

If it is a **project repo** (e.g. `site-v2`): set `base: "/site-v2/"`.

Then:
```bash
npm run deploy
```
